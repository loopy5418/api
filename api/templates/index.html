POST /image-with-text

This API requires an API key.

Request Body

{
  "image_url": "https://example.com/image.png",
  "text": "Hello World!",
  "position": "center", // or "top", "bottom", or [x, y] or "(x, y)"
  "color": "#FF0000",
  "font_size": 32,
  "font_style": "bold" // or "normal", "italic", "bold-italic"
}

Position Options

  • "top": Centered at the top of the image
  • "center": Centered in the middle of the image
  • "bottom": Centered at the bottom of the image
  • [x, y]: Custom pixel coordinates as an array (e.g., [50, 100])
  • "(x, y)": Custom pixel coordinates as a string (e.g., "(50, 100)")

Font Style Options

  • "normal": Regular font
  • "bold": Bold font
  • "italic": Italic font
  • "bold-italic": Bold and italic font

Response

Returns a PNG image with the specified text drawn on it.

GET /wifi-qr

This API requires an API key.

Query Parameters

  • ssid: WiFi network name (required)
  • password: WiFi password (optional, default: empty)
  • security: Security type (WPA, WEP, nopass; optional, default: WPA)
  • hidden: Whether the network is hidden (true/false; optional, default: false)

Example URL

https://api.loopy5418.dev/wifi-qr?ssid=MyWiFi&password=MyPass123&security=WPA&hidden=false

Response

Returns a PNG image of a QR code that allows devices to connect to the specified WiFi network.

GET /ai

This API requires an API key. 100% free.

Supported Models

  • gpt-4: GPT-4 from OpenAI
  • gpt-4o: Multimodal GPT-4 Omni from OpenAI
  • gpt-4o-mini: Lightweight GPT-4 Omni variant
  • o1: OpenAI's small model (experimental)
  • o3-mini: Miniature OpenAI model (possibly under o-series)
  • llama-3.1-8b: Meta’s LLaMA 3.1 model
  • llama-3.1-70b: Meta’s LLaMA 3.1 model
  • llama-3.2-11b: Hypothetical LLaMA 3.2
  • llama-3.3-70b: Hypothetical LLaMA 3.3
  • gemini-1.5-pro: Google’s Gemini 1.5 Pro model
  • gemini-2.0-flash: Google’s lightweight Gemini 2.0 Flash variant
  • deepseek-v3: DeepSeek's version 3 LLM
  • deepseek-r1: DeepSeek's first research-focused model

Query Parameters

  • model: The model of the AI you want to use. See supported models above. (required)
  • web_search: Use the web search feature to get the latest information online. Some of the models do not support this. (optional, defaults to false)
  • prompt: The prompt you want to send the AI. (required)
  • key: Your API key. You can get one from our Discord server by pressing the Discord button above (required)

Example URL

https://api.loopy5418.dev/ai?prompt=Get+the+latest+news+in+Europe&model=gpt-4o&web_search=true

Response Preview

{
  "success": true,
  "model": "gpt-4o",
  "response": "Hello! How may I assist you today?",
  "prompt": "Get the latest news in Europe"
}
