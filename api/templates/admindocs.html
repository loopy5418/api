Admin API

Base URL: https://api.loopy5418.dev/admin

super secret admin endpoint documentation

GET /adminsignin

Details

Sign in with your admin credentials. Browser only (currently does not work)

Query Parameters:

  • No query parameters.
POST /restart

Details

This endpoint restarts the worker dyno.

Headers:

  • X-API-KEY: string (your api key, required)

Example Response:

{
    "message": "Heroku dyno restart triggered!"
}
POST /evaluate

Details

This endpoint runs a command in the worker dyno.

Headers:

  • X-API-KEY: string (your api key, required)

Example Request Body:

{
  "cmd": "pip --version"
}

Example Response:

{
  "stdout": "pip 24.0 from /usr/local/lib/python3.11/site-packages/pip (python 3.11)\n",
  "stderr": "",
  "returncode": 0,
  "success": true
}
POST /admin/generate-key

Details

Generates a user API key (not admin).

Headers:

  • X-API-KEY: string (admin API key, required)

Request Body:

{
  "user_id": "1016311833382105100"
}

Example Response:

{
  "api_key": "XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX",
  "user_id": "1016311833382105100"
}
GET /admin/get-key

Details

Retrieves a user's API key.

Query Parameters:

  • user_id: string (required)

Headers:

  • X-API-KEY: string (admin API key, required)

Example URL:

https://api.loopy5418.dev/admin/get-key?user_id=1016311833382105100

Example Response:

{
  "api_key": "XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX",
  "user_id": "1016311833382105100"
}
