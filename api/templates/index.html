GET /currency-converter

Query Parameters

  • base: The base currency code (e.g., USD)
  • target: The target currency code (e.g., EUR)
  • amount: The amount to convert

Example URL

https://api.loopy5418.dev/currency-converter?base=USD&target=EUR&amount=10

Response Preview

{
  "base": "USD",
  "target": "EUR",
  "amount": 10,
  "converted": 9.25,
  "rate": 0.925
}
